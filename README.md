# Модель OCR для распознавания CAPTCHA

**Задача**: Построить модель через слои стандартной библиотеки torch, без использования torchvision.models или аналогов.

## Описание решения

Для решения поставленной задачи реализован полный OCR пайплайн, включающий реализованный класс данных, считывающий входные изображения и их метки из имён файлов, воспроизводимое разделение данных на выборки.

Создана и обучена модель архитектуры CRNN, состоящая из сверточной и рекуррентной частей:

- Сверточная сеть включает четыре блока Conv2D → BatchNorm → ReLU, между которыми расположены три слоя MaxPooling, последовательно уменьшающие пространственное разрешение изображения и увеличивающие число каналов с 3 до 192.
- Полученная карта признаков преобразуется в последовательность путем усреднения по вертикальной оси.
- Последовательность признаков обрабатывается двухслойной двунаправленной LSTM с размером скрытого состояния 128, что позволяет учитывать контекст как слева направо, так и справа налево.
- Затем с помощью слоя AdaptiveAvgPool1d длина последовательности приводится к фиксированному числу позиций, соответствующему длине CAPTCHA.
- На последнем этапе полносвязный слой (Linear) выполняет классификацию символов в каждой позиции.
- Обучение модели производится с использованием функции потерь CrossEntropyLoss, а оптимизация выполняется алгоритмом AdamW с динамическим уменьшением скорости обучения (ReduceLROnPlateau).

## Результаты тестов

- Лучший validation CER: 0.0187 на эпохе 41.
- Итоговая точность полного совпадения CAPTCHA: 92.52%.
- Итоговый CER на независимой test-выборке: 0.0159.
